🎓 Automated Contactless Attendance System

UPM Faculty of Engineering | EEE4948 Integrated Design Project

🚀 The Future of Classroom Attendance

Say goodbye to slow, inaccurate paper sign-ins and vulnerable proxy attendance! We have developed a state-of-the-art Automated Face Recognition Attendance System to identify students and record attendance contactlessly using edge computing.

⚙️ How It Works

  • Face Detection: Uses MTCNN to quickly locate faces.
  • Facial Recognition: Uses FaceNet to generate a 512-D face embedding to match enrolled students.
  • Liveness Detection: Uses an ONNX Anti-Spoofing Model to verify a real person is present.

✨ Key Features

  • ✅ Timetable Detection: Automatically classifies students as Present or Late based on the schedule.
  • 🛡️ Bulletproof Security: Failsafe against photos or digital screens with built-in anti-spoofing.
  • 💻 Web Dashboard: A dedicated admin portal backed by FastAPI and SQLite for seamless student registration and CSV logging.

📊 Project Results

Recognition Accuracy: 100%
Anti-Spoofing Success: 100%
Inference Speed: 30 FPS
Prototype Budget: RM 757
